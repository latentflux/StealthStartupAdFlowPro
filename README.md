# AdFlow Pro - Developer Credits

A modern landing page built with Next.js and TypeScript.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 15
- TypeScript  
- Tailwind CSS
- React
